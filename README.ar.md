# المترجم العربي المُؤَسَّس على المُدوَّنة

> 🇬🇧 [English README](README.md) — هذا الملف هو النسخة العربية، صياغةً بالعربية الفصحى لا ترجمةً حرفية.

مهارةُ ترجمةٍ من الإنجليزية إلى العربية (وبالعكس) **مؤسَّسةٌ على مُدوَّنةٍ عربيةٍ حقيقية**، لا على بيانات تدريب النموذج وحدها. تَعتمد اعتمادًا صلبًا على [`arabic-corpus-toolkit`](https://github.com/drxvb/arabic-corpus-toolkit) (الإصدار ≥ v1.13.0). مبنيةٌ على مواصفات [Agent Skills](https://agentskills.io).

## خطُّ المعالجةِ ذو الخمسِ مراحل

| المرحلة | الدور |
|---|---|
| **A — المصطلحات** | بحثٌ في قاموس المجموعة (٣٤٠ مدخلًا) + تأكيدُ الأصل F من المُدوَّنة + حقنُ أزواج EN↔AR المباشرة من الأصل G |
| **C — مسوَّدةُ النموذج** | نداءُ أيِّ نقطةِ نهايةٍ متوافقةٍ مع OpenAI، يَمُرُّ عبر عقد `safe_llm_call` المشترك |
| **D — المُدقِّق** | معدَّلُ الترجمة الحرفية + أمانةُ المصطلح + طبيعيةُ النَّظْم؛ تطبيعُ G1 عبر المجموعة؛ إعادةُ توليدٍ عند الفشل |
| **E — المراجعةُ عبر المزوِّدين** | مراجعةٌ ثانيةٌ تَلتقط هلوساتِ نموذجٍ واحد، مع تصحيحٍ آمنٍ للنطاقات الفرعية |
| **F — بوَّابةُ المؤنسِن** | فحصُ جودةٍ أسلوبي (`--quality-gate` تجريبي / `--deep-quality` بمعيارٍ معرفيٍّ رباعي الأبعاد) |

> **المرحلة B (ذاكرةُ الترجمة) أُسقِطَت نهائيًّا**: كشف فحصُ واقع المُدوَّنة (v0.1.1) أنَّ المحاذاةَ الثنائيةَ المفترَضةَ غيرُ موجودة (صفرُ مجلَّدٍ مُزدوَجٍ من ٧٧٬٧٥٦)؛ والأصلان G وF يُغنِيان عن ذاكرة الترجمة.

## الجديد في v1.9.0 — مرونةُ أعطال مزوِّدِ النموذج

تَمُرُّ المرحلةُ C الآن عبر عقد `safe_llm_call` من المجموعة v1.13.0: إعاداتُ محاولةٍ بتراجعٍ أُسِّي، وقاطعُ دارةٍ لكلِّ مزوِّد (ثلاثةُ إخفاقاتٍ ← فتحٌ ٦٠ ثانية)، ومُغلَّفُ عطلٍ منظَّم. تَتضمَّن نتيجةُ المرحلة C الآن `error_class` و`attempts` و`circuit_open` فيَستطيعُ المُستدعي التمييزَ بين العطل العابر والنهائي. ويَتراجع تلقائيًّا إلى مسار `urllib` المباشر حين تكون المجموعةُ أقدمَ من v1.13.0. كذلك تُتيح v1.8.0 مُرشِّحَ `min_consensus` لِحصرِ مصطلحات الأصل G في ما صادَق عليه أغلبيةُ المزوِّدين أو إجماعُهم.

## البداية السريعة

```bash
export LLM_API_URL=https://api.openai.com/v1/chat/completions
export LLM_API_KEY=sk-...
export LLM_MODEL=gpt-4o-mini

python scripts/translate.py --input article.en.txt --domain news --output article.ar.txt
```

أو اختر مزوِّدًا محليًّا مباشرةً عبر العَلَم `--llm-proxy {kimi,codex,gemini,minimax}`.

## التحقق من سلامة التثبيت

```bash
python evals/test_translator.py            # ١٦ — مجموعةُ المسار الأساسية
python evals/test_stage_contracts.py       # ٢٠ — مرونةُ المرحلة C + ترشيحُ min_consensus + حُكْمُ المرحلة D
python evals/test_terminology_and_trace.py # ١٩ — مصطلحاتُ المرحلة A + سجلُّ الأثر G3
```

جميعُها حتميةٌ بلا أيِّ نداءٍ حيٍّ لنموذجٍ لغوي.

## النطاق

مترجمٌ، **ليس** مؤنسِنًا ولا مُوَطِّنًا (localizer). **لا تَستخدمه** للترجمات اللهجية (الفصحى فحسب)، ولا للترجمة الأدبية الإبداعية، ولا لتوثيق الشيفرة (احفظ الإنجليزيةَ التقنية)، ولا للنصِّ العربيِّ أصلًا (استخدم المؤنسِن). تاريخُ الإصدارات الكاملُ في [`CHANGELOG.md`](CHANGELOG.md).
